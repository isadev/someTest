1.- Descagar el proyecto
2.1 - Crear base de datos en postgres
    CREATE DATABASE strap_test_db;
2.1.- Dar permisos a la carpeta
    sudo chown 'whoami':www-data -R "ruta_hacia_el_proyecto"/someTest/var/cache/
    sudo chmod -R 775 "ruta_hacia_el_proyecto"/someTest/var/cache/

    sudo chown 'whoami':www-data -R "ruta_hacia_el_proyecto"/someTest/var/logs/
    sudo chmod -R 775 "ruta_hacia_el_proyecto"/someTest/var/logs/
3.- Correr composer
    composer install
3.0 - Configurar el archivo parameters.yml (en caso de cambiar los introducido anteriormente) dentro de la carpeta
      	app->config->parameters.yml
      		cambiar los valores de la base de datos por la que se desee
3.1 - Correr los asstes
    bin/console  assets:install --symlink
4.- Correr las migraciones
	bin/console d:m:m
4.1- Correr el servidor
    bin/console server:run
5.- Ir al navegador
	http://127.0.0.1:8000/app_dev.php/home
