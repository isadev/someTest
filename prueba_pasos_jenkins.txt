Pasos para correr el jenkins

PASOS PARA CONECTARSE AL SERVIDOR DONDE SE HARAN LOS PULL DE LAS RAMAS
0.- En una instalacion sin docker 
	* sudo su - jenkins
1.- crear la llave publica en el servidor donde esta jenkins
    * ssh-keygen -t rsa (no generarle passphrase )
    * cat ~/.ssh/id_rsa.pub
2.- crear desde la maquina del jenkins el enlace hacia la maquina que mantendra los datos del repositorio
	* ssh-copy-id usuario_maquina_repositorio@ip_maquina_repositorio
3.- crear denro de la maquina del repositorio, el archivo .sh que tendra las instrucciones de correr tanto el docker
